# Da Nang · Hoi An Family Trip 2026

2026년 5월 28일 ~ 6월 2일 다낭·호이안 가족여행 아카이브입니다.

## 보기

- 메인 여행 계획서: [`index.html`](./index.html)
- 고정 컨텍스트: [`docs/trip-context.md`](./docs/trip-context.md)
- 버전 보관: [`versions/`](./versions/)

## 여행 핵심

- 구성: 부부 + 약 300일 아기 + 부모님
- 원칙: 아기 컨디션 최우선, 부모님 무리 금지, 12~15시 더위 회피
- 방식: 후보는 넉넉히 두되 현장에서는 과감히 삭제
- 이동: 차량/밴 적극 활용

## 현재 문서 상태

- 문서 목적과 사용자 요청 기반 버전/변경 이유를 HTML에 포함
- 버전/요청 이력은 HTML 최상단 접이식 패널로 제공
- 식당/카페에서 아기의자·조용한 자리·스피커에서 떨어진 자리 등을 요청하는 아기 동반 영어회화 섹션 포함
- 맛집·카페·마사지는 네이버 VIEW/블로그/카페 2025~2026 노출 신호와 가족 운영성을 함께 반영해 v16에서 전면 재분류, v18에서 5/29 점심을 La Plage에서 Tân Phát/탄팟 중심으로 재조정
- 네이버카페 커뮤니티 후기 신호와 블로그 후기 신호를 구분
- Google Maps/공식 확인 태그 기반 후보 분류 포함
- 네이버 한국어 후기 빠른 점검 포함
- 네이버픽 추가 후보 포함
- 티엔킴·템하이산 계열 네이버/방송·연예인픽 후보를 5/31·6/1 동선에 반영
- 5/30 토요일·5/31 일요일 저녁식사와 카페를 금요일처럼 시간대별로 상세화
- 5/29 금요일 올드타운 저녁 후보(해산물식당·카페) 추가
- 5/30 토요일 올드타운 중복 방문 삭제: 수영·점심·카페·마사지·저녁식사+카페 중심으로 수정
- 출국/귀국 항공 및 숙소 고정값은 `docs/trip-context.md`에서 관리

## 업데이트 규칙

고정값이 바뀌면 먼저 `docs/trip-context.md`를 수정한 뒤 `index.html`을 업데이트합니다.
